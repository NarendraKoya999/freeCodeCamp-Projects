JavaScript and Java

JavaScript and Java are similar in some ways but fundamentally different in some others. The JavaScript language resembles Java but does not have Java's static typing and strong type checking.

Variables

You use variables as symbolic names for values in your application. The names of variables can be used to store values for later use.

var x = 10;

Declaring variables

You can declare variables on two different occasions. In the first case, the variable is declared upon assignment of a value to it.

var x = 10;

Variable scope

When you declare a variable outside of any function, it is called a global variable, because it is available to any part of the program.

When you declare a variable within a function, it is called a local variable, because it is available only within that function.

if (true) { var x = 10; } console.log(x); // Prints "10" if (true) { var x = 10; } // ReferenceError: x is not defined if (true) { let x = 10; } console.log(x); // Prints "10"

Global variables

Global variables are in fact properties of the global object. In web pages the global object is window, so you can set and access global variables using the window.variable syntax.

var carName = "Volvo"; var x = 10; carName; // Prints "Volvo" x; // Prints "10" var carName = "Volvo"; var x = 10; carName; // Prints "Volvo" x; // Prints "10" var carName = "Volvo"; var x = 10; window.carName; // Prints "Volvo" window.x; // Prints "10" var carName = "Volvo"; var x = 10; delete carName; carName; // Prints "undefined" x; // Prints "10" var carName = "Volvo"; var x = 10; delete window.carName; carName; // Prints "undefined" x; // Prints "10" var carName = "Volvo"; var x = 10; delete window.x; carName; // Prints "Volvo" x; // Prints "10" var carName = "Volvo"; var x = 10; delete window.x; delete window.carName; carName; // Prints "undefined" x; // Prints "10" var carName = "Volvo"; var x = 10; delete window.x; delete window.y; carName; // Prints "undefined" x; // Prints "10" var carName = "Volvo"; var x = 10; delete window.x; delete window.y; delete carName; carName; // Prints "undefined" x; // Prints "10" var carName = "Volvo"; var x = 10; delete window.x; delete window.y; delete window.carName; carName; // Prints "undefined" x; // Prints "10" var carName = "Volvo"; var x = 10; delete window.x; delete window.y; delete window.carName; delete x; carName; // Prints "undefined" x; // Prints "10"

Constants

You can create a read-only, named JavaScript object. In other words, you can create a constant with the const keyword. You cannot reassign the value of a constant declared with const.

const cars = ["Saab", "Volvo", "BMW"]; cars[0] = "Opel"; cars; // Prints ["Opel", "Volvo", "BMW"] const cars = ["Saab", "Volvo", "BMW"]; cars[0] = "Opel"; cars; // Prints ["Opel", "Volvo", "BMW"]

if...else statement

Use the if...else statement to specify different actions for different conditions.

if (condition) { statement_1; } else { statement_2; } if (condition) { statement_1; } else if (condition) { statement_2; } else { statement_3; } if (condition) { statement_1; } else if (condition) { statement_2; } else if (condition) { statement_3; } else { statement_4; } if (condition) { statement_1; } else if (condition) { statement_2; } else if (condition) { statement_3; } else if (condition) { statement_4; } else { statement_5; }

while statement

A while statement executes as long as a specified condition is true.

while (condition) { statement; } let i = 0; while (i < 10) { console.log(i); i++; }

Function declarations

A function declaration can be hoisted (hoisted to the top of its scope) or be lazily evaluated.

function myFunction(a, b) { return a * b; } myFunction(5, 5); // Prints "25"
